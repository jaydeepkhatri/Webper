### Installation

1. Clone the repo
```sh
git clone https://github.com/jaydeepkhatri/Webper.git
```
2. Change folder
```sh
cd Webper
```
3. Install NPM packages
```sh
npm install
```
4. Start the server
```sh
npm start
```
